# 📚 Self-Learning Through Documentation & Forums

## 🚀 Objective

This repository documents my journey of learning and building projects **without using tutorials or AI assistance**.

I am committed to:

* 📖 Learning only from official documentation
* 💬 Using community forums (Stack Overflow, ROS Discourse, GitHub Issues, etc.)
* 🧠 Understanding concepts deeply instead of copying solutions
* 🛠 Building projects from scratch

---

## 🎯 Why This Approach?

In robotics and systems engineering, real-world problems do not come with step-by-step tutorials.

By relying only on documentation and technical discussions, I aim to:

* Develop strong problem-solving ability
* Improve debugging skills
* Build documentation-reading habits
* Think like an engineer, not a tutorial follower

---

## 📌 Learning Rules

1. No YouTube tutorials
2. No course-based walkthroughs
3. No AI-generated solutions
4. If stuck:

   * Read documentation again
   * Search official forums
   * Read source code if necessary
   * Debug systematically
